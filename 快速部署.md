# 🚀 HEIGO 联赛管理系统 - 快速部署指南

## 方式一：Docker 部署（推荐）

```bash
cd /home/admin/openclaw/workspace/heigo-league-pro

# 1. 构建镜像
docker build -t heigo-league-pro .

# 2. 运行容器
docker run -d \
  --name heigo-league-pro \
  -p 5000:5000 \
  -e SECRET_KEY="heigo-secret-$(openssl rand -hex 16)" \
  -v $(pwd)/data:/app/instance \
  heigo-league-pro

# 3. 查看日志
docker logs -f heigo-league-pro

# 4. 访问
# 浏览器打开：http://localhost:5000
```

**默认账号：**
- 用户名：`admin`
- 密码：`admin123`

## 方式三：Docker Compose（推荐生产环境）

```bash
cd /home/admin/openclaw/workspace/heigo-league-pro

# 启动
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止
docker-compose down
```

## 💾 数据备份

数据保存在 `./data/` 目录，定期备份：

```bash
# 备份数据库
cp -r data/ backup_$(date +%Y%m%d)/

# 恢复数据
docker cp backup_$(date +%Y%m%d)/heigo-league-pro:/app/instance
```

## ⚠️ 注意事项

1. **首次运行** 会自动创建数据库和管理员账号
2. **密码修改** 登录后请立即修改默认密码
3. **数据持久化** 使用 `-v` 挂载数据卷，防止数据丢失
4. **端口占用** 如果 5000 端口被占用，修改 `-p 5001:5000`

## 🆘 遇到问题？

### 端口被占用
```bash
# 修改端口
docker run -d -p 5001:5000 ...
```

### 权限问题
```bash
# 给目录权限
chmod -R 777 data/
```

### 镜像构建失败
```bash
# 清理缓存
docker system prune -a
# 重新构建
docker build --no-cache -t heigo-league-pro .
```
